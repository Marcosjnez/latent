# Print Method for Latent Class Diagnostics

Print Method for Latent Class Diagnostics

## Usage

``` r
# S3 method for class 'lclassd'
print(x, ...)
```

## Arguments

- x:

  An object of class `"lclassd"` returned by
  [`lclass_diag()`](https://marcosjnez.github.io/latent/reference/lclass_diag.md).

- ...:

  Additional arguments passed to `print.default` or `print.data.frame`.
