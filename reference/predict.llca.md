# Predicted values for Latent Class Analysis.

Get predicted class membership probabilities for latent class models.

## Usage

``` r
predict(model, new = NULL)
```

## Arguments

- model:

  Fitted llca object.

- new:

  Matrix or data.frame of new predictors.

## Value

Stuff:

- dof:

  Degrees of freedom
