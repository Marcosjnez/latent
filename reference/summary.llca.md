# Fit indices

Compute fit indices from any model.

## Usage

``` r
getfit(model)
```

## Arguments

- model:

  data.frame or matrix of response.

## Value

List with the following fit indices:

- AIC:

  .

- BIC:

  .

## Details

`getfit` computes all the fit indices related to a specific model.
