# Inspect objects from fitted lca models.

Inspect objects.

## Usage

``` r
latInspect(fit)
```

## Arguments

- fit:

  model fitted with lca.

- digits:

  Number of digits to print.

## Value

List with one of the following objects:

- class:

  .

- posterior:

  .

- profile:

  .

## Details

Extract objects from fitted lca object.
