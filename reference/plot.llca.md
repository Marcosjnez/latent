# Plot regression coefficients

Plot the coefficients of latent models.

## Usage

``` r
plot(fit)
```

## Arguments

- fit:

  model fitted with lca.

- confidence:

  Coverage of the confidence interval.

## Value

List with the following objects:

- vcov:

  Variance-covariance matrix between the parameters.

- se:

  Standard errors.

- SE:

  Standard errors in the model list.

## Details

Obtain a plot with the coefficients of latent models.
