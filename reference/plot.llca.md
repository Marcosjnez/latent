# Plot method for latent class analysis (llca) objects

Plot method for latent class analysis (llca) objects

## Usage

``` r
# S3 method for class 'llca'
plot(x, ...)
```

## Arguments

- x:

  An object of class `"llca"`, typically from `latent`.

- ...:

  Additional arguments passed to `plot_lca_mixed`.

## Value

A list of ggplot objects (as from `plot_lca_mixed`).
