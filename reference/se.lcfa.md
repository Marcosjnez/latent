# Standard Errors

Compute standard errors.

## Usage

``` r
se_cfa(fit)
```

## Arguments

- fit:

  model fitted with lca.

- confidence:

  Coverage of the confidence interval.

## Value

List with the following objects:

- vcov:

  Variance-covariance matrix between the parameters.

- se:

  Standard errors.

- SE:

  Standard errors in the model list.
